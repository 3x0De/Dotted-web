# Dotted-web

Ce repo fait partis du projet [Dotted](https://github.com/3x0De/Dotted-docs/)

## FEATURES

- Visualisation des projets disponnibles par utilisateurs
- Gestion de la visibilité des projets
- Modification des blocs des éléments sur une page
- Drag and drop pour organiser les projets
- Lien avec la base de donnée
- Gestion des logins

## Installation

Tout est détailé [ici](https://github.com/3x0De/Dotted-docs/blob/main/INSTALLATION.md).
Nécécite le backend pour fonctionner.
